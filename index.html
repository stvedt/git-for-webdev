Useful Terminal Commands

cd - Change Directory

ls <fileName> - List Files

rm <fileName> - remove file

mc - move

mkdir - Make Directory

help - help

Git Config Commands

git config --global user.name "Your Name"

git config --global user.email "your@email.com"

Add some color

git config --global color.diff auto

git config --global color.status auto

git config --global color.branch auto

Initialize Git (Locally)

Do a CD to the directory you want to start using Git for.

cd <nameOfDir>

git init

Creating Versions & Committing

Create Version

Step 1 - Stage
Step 2 - Commit

Note: If a file has been modified after being staged it must be restaged for the changes to be committed.

git add <file(s)>
git commit -m 'commit message here'

-m , commit message
-a , add all, adds and commits all files

Modifying staged files

How to view the status, difference and remove from stage.

git status -s

git diff

git reset HEAD <fileName>

-s, displays a condensed (short) version of status

Ammending a Commit

git commit --amend -m 'optional overwrite of original message'

More information/modification

Diplay Logs of Commits

git log

git log -p -2

git log --stat

-p, includes the file diff's with logs
-2, displays the two most recent logs (use any num)
--stat, abbreviated version of logs

Get a Clone of Git Repository

git clone <git@github.com:user/repoName.git>
